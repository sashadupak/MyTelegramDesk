## Моя стратегия поиска
Я НЕ ищу вакансию. Я ищу LEVERAGE-ТОЧКИ — места где мой нелинейный профиль даёт ×10 ценности.

### A. Role Carving (вырезание роли под себя)
Нахожу компанию где: есть деньги, есть хаос/новая зона, нет человека который это держит → предлагаю себя как решение.

### B. Opportunity-driven search
Не "кем работать", а: где сейчас появляется новая возможность/рынок/технология?
AI-agents, дроны/автономка, новые форматы ивентов, корпоративные инновации.
Захожу раньше чем появляются вакансии.

### C. Platform Entry
Захожу не в "роль", а в платформу (экосистема, фонд, lab, R&D центр) где можно запускать разные направления.

## Scoring: 6 критериев оценки вакансии (каждый 0-5)

### 1. Свобода действий
Сигналы: "launch", "build", "new direction", "innovation", "0→1", "с нуля"
0 = жёсткие рамки, execution only | 5 = полная автономия, создание нового

### 2. Доступ к ресурсам
Корпорация / фонд / lab, есть ли бюджет, команды, инфраструктура
0 = нет ресурсов, "работа за идею" | 5 = серьёзный бюджет и поддержка

### 3. Близость к top management
Кому репортит? Есть ли interaction с C-level / фаундерами?
0 = 3+ уровня до CEO | 5 = напрямую CEO/CTO/фаундер

### 4. Неопределённость роли (ЭТО ПЛЮС для меня)
Размытое описание, нет чётких рамок — значит можно "захватить" роль
0 = жёсткий JD, checklist задач | 5 = "мы сами не знаем что нужно, ищем человека"

### 5. Соответствие драйверам
Инновации, AI, ивенты, новые продукты, спорт/медиа, deep tech
0 = скучная индустрия без инноваций | 5 = прямое попадание в мои интересы

### 6. Возможность создать новое направление (САМЫЙ ВАЖНЫЙ)
Можно ли из этой роли вырасти в отдельный юнит / продукт / бизнес?
0 = потолок виден сразу | 5 = роль может вырасти в бизнес

## Red Flags (жёсткий стоп, не показывать)
- "поддержка существующего продукта"
- "execution only"
- "strict processes / reporting"
- нет слов про growth / innovation
- жёстко регламентированные роли (support, ops без свободы)
- узко-исполнительские позиции
- без бюджета / влияния
